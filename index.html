Ideas Principales.

  • La propiedad de posicion especifica el tipo de metodo de posicionamiento utilizado para un elemento
  • Hay cinco valores de posicion diferentes:Los elementos se posicionan usado las poropiedades top, bottom, left y right.
  • Tambien funcionan de manera diferente dependiendo del valor de la posicion.

Apuntes de Clase.

Position relative:

Un elemento con position; esta posicionado en relacion a su posicion normal. El posicionamiento relativo desplaza un elemento respecto de su posicion original.

Position absoluto:

El posicionamiento absoluto se emplea para establecer de forma exacta la posicion en la que se muestra la caja de un elemento

Position: fixed;

Un elemento con position: fixed; esta ubicado en relacion con la ventana grafica, lo que significa que siempre permanece en el mismo lugar.

Position sticky;

Es un tipo de posicionamiento que permite que un elemento se comporte de dos maneras diferentes según el desplazamiento del usuario

Resumen.

Las propiedades position nos permite generar diseños mas atractivos para los usuarios sin necesidad de utilizar otros leguajes de programacion como puede ser java
